Guide

This is a guide to using this academic portfolio template. You can customize it to fit your needs.

Getting Started

  1. Fork this repository on GitHub
  2. Edit the HTML files to add your personal information
  3. Update the CSS to customize the look and feel
  4. Add your publications, talks, teaching experience, and other content
  5. Deploy your site using GitHub Pages

Customization Options

You can customize various aspects of this template:

  • Color scheme
  • Layout and structure
  • Fonts and typography
  • Add or remove sections as needed

Adding Content

To add new content (publications, talks, etc.), simply edit the HTML file and add new items following the existing structure.

Need Help?

If you need assistance with this template, you can:

  • Check the documentation
  • Open an issue on GitHub
  • Contact the template maintainer
